# FOSS Club — Tech Interview 2026

Submitted by: Nivya Sudeesh

A static website for the FOSS Club, built as a Tech Interview 2026 project.


## Project Overview

This repository contains a simple landing page for the FOSS Club. The site includes:

- `index.html` — main homepage with event cards and contact section
- `register.html` — registration placeholder page
- `style.css` — styling for the site
- `images/` — image assets used by the page

## Notes

- The register page currently shows a "coming soon" message.
- The project is implemented with HTML and CSS only.


## Workflow: How to Clone and Contribute:

1. Fork the Repository
->Go to
```bash
git clone https://github.com/your-username/repo.git
```
Click the Fork button (top-right) to copy it to your own GitHub account.




2. Clone to Your Local System
```bash
git clone https://github.com/<your-username>/tech_interview_2026.git
```
Navigate into the project folder:
```bash
cd tech_interview_2026
```



3. Make Changes
Open the project in VS Code (or any editor):
```bash
code .
```



4. Add, Commit, and Push
Stage all your changes:
```bash
git add .
```
Write a clear, descriptive commit message:
```bash
git commit -m "addED contact section, event cards, and register page AND changed UI of the page"
```
Push to your forked repository:
```bash
git push origin main
```



5. Deploy with GitHub Pages

1.Go to your forked repository on GitHub.    
2.Click Settings → Pages (left sidebar).    
3.Under Source, select branch main and folder / (root).  
4.Click Save.

Your live site will appear at:
```bash
https://nivyasudeesh.github.io/tech_interview_2026/
```
It may take 1–2 minutes to go live after saving.




6. Create a Pull Request

1.Go to your forked repo on GitHub.     
2.Click Compare & pull request (banner at the top).      
3.Add a clear title and description of what you changed.      
4.Click Create pull request
